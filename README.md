# Task Management

A React task management application built with Vite.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Install Ant Design

Ant Design is already included in the project. If you need to install it:

```bash
npm install antd
```

### Install Tailwind CSS

```bash
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

Then add Tailwind directives to your CSS file (e.g., `src/index.css`):

```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

### Run the Project

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
